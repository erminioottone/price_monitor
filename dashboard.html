🚀 Setup Guide — 10 minutes, one-time
1
Create a GitHub repository
Go to github.com/new → create a new private repo named price-monitor.
2
Upload the project files
Unzip the downloaded archive. In GitHub, click Add file → Upload files and upload all files keeping the folder structure.
Required: src/monitor.py · products.json · prices.json · requirements.txt · .github/workflows/price-monitor.yml
3
Add 5 repository secrets
Settings → Secrets and variables → Actions → New repository secret:

SMTP_HOST → smtp.gmail.com
SMTP_PORT → 587
SMTP_USER → your Gmail address
SMTP_PASS → Gmail App Password (step 4)
NOTIFY_EMAIL → email to receive alerts

Outlook users: SMTP_HOST = smtp.office365.com
4
Create a Gmail App Password
1. Enable 2FA on your Google account.
2. Go to myaccount.google.com/apppasswords
3. Generate for "Mail" — use the 16-char code as SMTP_PASS.
5
Add products using this dashboard
Click + Add Product, fill in the details, then click Copy to clipboard under the JSON section. In GitHub, open products.json, click the pencil icon, paste and commit.
6
Test manually
In GitHub: Actions → Price Monitor → Run workflow. Check logs to confirm everything works.
